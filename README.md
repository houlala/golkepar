# Golkepar

Scripts:
### download-and-update-geo-ip.py
To download/update geolite2 files.

### download-ovh-logs.py
To download last Apache logs from OVH Web Hosting.
 
### ingest-logs-to-parquets.py
To parse logs into optimized parquets files with geo IP informations from geolite2.

### web-stats-duckdb.py
To ingest parquets files into a DuckDB file and create usefull tables for analytics.
